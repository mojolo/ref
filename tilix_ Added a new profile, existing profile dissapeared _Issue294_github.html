I think I found the issue. The ID of my profile above (the one that got removed) is 2b7c4080-0ddd-46c5-8f23-563fd3ba789d, that is the hard coded default ID of the first profile created by terminix.

When adding a new profile this gets removed, see https://github.com/gnunn1/terminix/blob/82ea3a78cc2f449112158a250a2d675f1a6110bd/source/gx/terminix/preferences.d#L206

What exactly was the idea behind this code? From what I can see it should be safe to drop this. Otherwise it leads to the pre-configured default profile to always get lost once you add a new one.
