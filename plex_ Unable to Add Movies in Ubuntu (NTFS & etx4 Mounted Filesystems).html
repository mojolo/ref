ChuckPaTeam Member
Feb '17

if /disks/internal/Movies is the mount point, PMS only needs the permission to read and traverse (r-x) of the Movies directory and all below it. You can always retain ownership. If it’s ext4, and internal, no special mount options are needed.

I would always keep yourself as the owner of your files. I never deviate from that.

I would suggest this to handle Movies

sudo chmod 775 /disks/internal/Movies

cd /disks/internal/Movies
sudo chown -R  your_linux_username .      # if you're not the current 'owner'
sudo find . -type d -exec chmod 775 {} \;
sudo find . -type f -exec chmod 664 {} \;

These instructions are based on having the 775 permissions we established earlier for /disks/internal/plexmediaserver.
