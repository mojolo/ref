[pyqt4/5] MKV Extractor Qt


Hello,

This version in Qt follows MKV Extractor Gui using G2S , the code is cleaner and requires less dependencies.

The software is in French and English.


Changelog :

Version 5.5.0 wrote:

  [english]
    - Bug corrections
    - Replacing the -I argument of mkvmerge with -J


  [French]
    - Fixed bugs
    - Replacing argument of -I -J by mkvmerge


To recall, the software can:
Extract tracks contained in files Matroska (.mkv) .
Convert audio files DTS to AC3 so that the file Matroska (.mkv) to be readable on the Freebox v5. (with possible amplification, switching to stereo)
Re-encapsulate a file Matroska (.mkv) after removing the tracks or the conversion of audio files DTS.
Check the conformity of the file Matroska (.mkv) via mkvalidator.
Optimize file Matroska (.mkv) via mkclean.
View attachments with the file Matroska (.mkv) .
Automatically convert a file srt vobsub via tesseract with multi cpu.


Sources:
Sources of the latest version : click build.sh (requires build dependencies)

Installation:
Via mon eg:

sudo add-apt-repository ppa:hizo/mkv-extractor-gui
sudo apt-get update

Version QT4:
Dépendances : mkvtoolnix, python3, python3-pyqt4

sudo apt-get install mkv-extractor-qt

or
- Latest version
- Version precise : is not updated but still works.

Version QT5:
Dépendances : mkvtoolnix, python3, python3-pyqt5

sudo apt-get install mkv-extractor-qt5

or
Last Version


Use:
Via the shortcut on the main menu (installation via PPP)
Via a "right click" on a mkv file and then "open with ..." (installation via PPP)
Via the command "mkv-extractor-qt" [mkv file] (installation via PPP) => mkv-extractor-QT5 to the latest version
via the launch of the python file MKVExtractorQt.py => MKVExtractorQt5.py to the latest version


Mkv File:
For those who have no mkv files on hand but want to try, here is a file open test (do not test the DTS conversions => ac3 and sub => SRT).


License:
The software is under GPL v3.


It is advisable to have:
ffmpeg or avconv (SDR => AC3)
mkclean (optimization of mkv file)
mkvalidator (tests mkv file)
qtesseract5 (SUB => SRT) (on my ppp): https: //forum.ubuntu -fr.org/viewtopic.php?pid=21507283
