This is intended for people not understanding why the configure option is not available after installing "Force Blur".

Apparently the owner of the package forgot to mention that it's required from the user to execute two commands, which are mentioned below, for the configure button to appear next to the about button.

Execute the following commands:
1. mkdir -p ~/.local/share/kservices5/
2. cp ~/.local/share/kwin/scripts/forceblur/metadata.desktop ~/.local/share/kservices5/forceblur.desktop
