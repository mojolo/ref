Main Gentoo repository (Portage tree)

Text and office apps

  • app-editors/focuswriter - fullscreen and distraction-free word processor
  • app-editors/juffed - tabbed text editor with syntax highlighting
  • app-editors/notepadqq - text editor similar to Notepad++
  • app-editors/tea - small, lightweight text editor
  • app-editors/qxmledit - Qt XML editor
  • app-office/lyx - frontend for LaTeX, DocBook, etc.
  • app-office/scribus - desktop publishing and layout program
  • app-office/texmaker - frontend for LaTeX
  • app-office/texstudio - Free cross-platform LaTeX editor (fork from texmakerX)
  • app-text/calibre - ebook management
  • app-text/coolreader - ebook reader
  • app-text/fbreader - ebook reader
  • app-text/goldendict - dictionary lookup program
  • app-text/kchmviewer - feature rich chm file viewer
  • app-text/qpdfview - tabbed PDF viewer

Multimedia

  • media-gfx/nomacs - image viewer
  • media-gfx/photoqt - Simple but powerful Qt-based image viewer
  • media-gfx/phototonic - image viewer
  • media-sound/audacious - audio player
  • media-sound/bempc - Music Player Daemon client with experimental UI
  • media-sound/cantata - featureful Music Player Daemon client
  • media-sound/clementine - modern music player and library organizer
  • media-sound/coquillo - audio tagger
  • media-sound/flacon - extracts audio tracks from audio CD image to separate tracks
  • media-sound/qastools - ALSA mixer GUI
  • media-sound/qmmp - Audio player with Winamp/XMMS skins support
  • media-sound/quimup - Music Player Daemon client with cover art display
  • media-video/avidemux - video editor/transcoder
  • media-video/baka-mplayer - libmpv-based media player
  • media-video/minitube - YouTube client
  • media-video/smplayer - great MPlayer/mpv frontend
  • media-video/vlc - media player

Network apps

  • mail-client/trojita - IMAP e-mail client
  • net-dialup/cutecom - serial terminal
  • net-im/psi - instant messenger
  • net-im/vacuum - Jabber client
  • net-irc/kvirc - Advanced IRC Client
  • net-misc/fatrat - download/upload manager
  • net-news/quiterss - RSS/Atom feed reader
  • net-news/rssguard - RSS/Atom feed reader
  • net-p2p/qbittorrent - BitTorrent client
  • net-p2p/transmission - BitTorrent client
  • net-wireless/wpa_supplicant - Wi-Fi front and back end (see also wpa_supplicant)
  • www-client/falkon - Web browser
  • www-client/otter - Web browser
  • www-client/qupzilla - Web browser
  • www-client/qutebrowser - Web browser

Desktop utilities

  • app-admin/keepassxc - password manager
  • sci-calculators/speedcrunch - calculator
  • sys-apps/qdirstat - Qt application analyzing disk space allocation
  • x11-apps/copyq - clipboard manager with advanced features
  • x11-misc/qlipper - clipboard history applet
  • x11-misc/qps - process manager
  • x11-misc/qxkb - keyboard layout switcher
  • x11-terms/qterminal - multitab terminal emulator
  • x11-misc/qtfm - lightweight file manager

Desktop environment

  • lxqt-base/lxqt-meta - lightweight desktop environment (see also LXQt)

Other

  • app-misc/anki - flash card program for learning purposes
