ObiWanTwo commented Jan 20, 2019

As far as I know there is no general option to disable transcoding once and for all, however you can do this in your profile settings. You should uncheck the following options:

  • Alow audio playback that requires transcoding
  • Allow video playback that requires transcoding
  • Allow media downloading and syncing that requires transcoding
