Get Youtube API Key

Jump to bottom
Daniel Ravina edited this page on Jan 14, 2020 15 revisions

Due to changes in the calculation and limits of the Youtube API combined with the significant growth Headset experienced in the last year, we are no longer able to provide a single API Key to all of our users. As a result, each user will have to provide their own API keys. This is a one-time, straight forward process which will allow you to continue using Headset without interruptions.

Important: You need to have a Gmail account to use Headset. If you need to create one, please go to this link https://myaccount.google.com/intro and click "Create a Google Account"


If you have any trouble with this guide. please email us at hello@headsetapp.co

Step 1/3: Sign in

a. Go to https://console.developers.google.com and sign in with your Gmail account:

Step 2/3: Create a project

a. In the console, click on Select a Project and then Start a New Project

Enlarge

b. Click CREATE


Enlarge

Step 3/3: Enable API

a. Click on "ENABLE APIS AND SERVICES" Enlarge

b. Search for "youtube" and select YouTube Data API v3 Enlarge

c. Click on ENABLE Enlarge

d. Click on Credentials on the sidebar menu, click on CREATE CREDENTIALS and then API Key Enlarge

e. Copy the generated API key and paste into the app and click SAVE Enlarge

That's it! You can now close the Google console and enjoy headset without any interruptions

Troubleshoot:

if you get an "Invalid Key" error with a newly created key, please wait about 5 minutes and try again. In addition, make sure that the API is generated from a new project. if you are using an existing Google Cloud project, the key will not work. please see https://github.com/headsetapp/headset-electron/issues/279 for more information.

Disclaimer:

DO NOT SHARE THE API KEY WITH ANYONE.

The key is saved on your computer and only used when communicating with Youtube. If you use Headset on more than one computer, you will have to repeat this step.

If you have any trouble with this guide. please email us at hello@headsetapp.co
