Hi!

I used to set my mouse speed with the option "resolution" in /etc/x11/in xorg.conf. A higher resolution increases the mouse speed.


Code:
Section "InputDevice"

# **********************************************************************
# Core Pointer's InputDevice section
# **********************************************************************

# Identifier and driver

    Identifier	"usb_mouse"
    Driver	"mouse"
    Option "Protocol"    "IMPS/2"
    Option "Device"      "/dev/input/mice"
    Option "Buttons" "5"
    Option "ZAxisMapping" "4 5"
    Option "Resolution" "256"
EndSection
