MKV Extractor GUI

This software is a graphical user interface ( GUI ) for extracting tracks from a file mkv or Matroska simple way following the Version 3 .

This software is licensed under the Creative Common BY-NC-SA.

It was written by hizoka through Glade2Script Bash and allowing freedom of action on the graphics. For more information, questions, comments, post a message here . This software is not part of the official repositories, its use is at your own risk.

Prerequisites

  • Have the administrative rights .
  • Have a configured and enabled Internet connection.
  • Having installed the package mkvtoolnix, ffmpeg, gettext, sed, bash, desktop-file-utils python-vte, python-gtksourceview .
  • The recommended outbuildings mkclean, mkvalidator, imagemagick, python-webkit, python-appindicator and mplayer2

These dependencies add many opportunities to the software (display more file types, verification and optimization of the Matroska file).

Installation

Via the PPP

  • Add the deposits of Hizoka : ACH: hizo / mkv-extractor-gui
  • Reload the package list .
  • Install the package mkv-extractor-QT5 .

In command line: 

sudo add-apt-repository ppa:hizo/mkv-extractor-gui
sudo apt-get update
sudo apt-get install mkv-extractor-qt5

Via the project .tar.gz

This version does not require installation.

Just download the archive to the following address: http://hizo.fr/linux/mkv_extractor_gui/MKV-Extractor-GUI.tar.gz , and extract its contents.

use

launching

The software launch is:

  • via a click on the file go_mkv_extractor_gui.sh
  • via the launcher in the main menu
  • via the control MKV-extractor-gui
  • by right-clicking on a mkv file if the option menu Service / Nautilus script is enabled in the software

Selecting the .mkv file

Loading file mkv is:

  • In selecting via the selection bar (only Matroska files are displayed).
  • Dragging on the selection bar.
  • Via a right click on a mkv file if the option menu Service / Nautilus script is enabled in the software.

Once the selection is made, the track list and some information appear in a list.

Select the output folder

This folder will be used to extract the tracks or the new file mkv .

By pressing the button next to the selection bar of the output file, the file folder mkv will be used.

Extract tracks mkv file

The track selection is via a click on the check box of the first column.

The selection releases the button Run and check Re-wrap the tracks .

Selecting a DTS audio file unlocks the check Convert DTS to AC3 .

You can re-encapsulate mkv files on and off the slopes extracted by checking the corresponding box.

The button Run then starts the work, a progress bar informs about the progress of it.

The work may at any time be canceled by clicking Cancel .

Check and repair the mkv file

It is possible to execute commands mkclean and mkvalidator the file mkv open.

These packages are dependencies MKV Extractor Gui that are installed if the software was installed by the PPP .

In the case of using the tar.gz version, it is the user to install (via my ppp the example ...)

mkvalidator verifies compliance specifications file mkv .

mkclean cleans and optimizes the file mkv .

The checkmark Replace the original file with the repaired file will tell mkclean that we must not create a new optimized file.

View attachments mkv file

You can view attachments types:

  • Text (including tags and chapter).
  • Image (jpg, bmp, png other).
  • Audio et Video (avi, mkv, mp3, ogg…).
  • Pages web (html).

Once the attached file is selected, the software offers the visualization of information on it or open it with your default software.

Configuration

The software offers various options:

  • File conversion settings DTS to AC3.
  • A language management using currently 3 languages ​​(EN, FR and CZ).
  • A removal of the extracted files in a re-encapsulation.
  • Using a Nautilus script / service menu to launch the software by right-clicking on mkv files.
  • A debug system further in case of trouble.

Preferences, files mkv file and output folder are retained in memory to load directly to the next using the software.

Update

  • Installation from PPP : The updates are done automatically via the usual package manager for the user.
  • Installing from the tar.gz files: This is the user to download the new version and manage new dependencies.

Uninstalling

If the software has been installed since its PPA

  • To simply remove the software, simply uninstall the package mkv-extractor-gui:
sudo apt-get remove mkv-extractor-gui
  • To uninstall the software and its configuration files , you must purge the package mkv-extractor-gui:
sudo apt-get purge mkv-extractor-gui
  • It is possible to remove the deposit Software:
sudo add-apt-repository --remove ppa:hizo/mkv-extractor-gui
  • It is also possible to remove the deposit and its uninstall packages :
sudo ppa-purge ppa:hizo/mkv-extractor-gui

If the software has been installed since tar.gz

  • To remove the software, simply delete the file.
  • To remove its configuration files, delete the file "$ {HOME} /. Config / mkv-extractor-gui /" .
