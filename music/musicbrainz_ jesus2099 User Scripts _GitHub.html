↑↑↓↓←→←→BA（コナミコマンド）

ARE YOU RAYDEE TO AUGMENT YOUR BORWSING ? (with userjs, user javascript, greasemonkey, userscripts, etc.)

YES ?

Here are my small Multi‐browser user scripts. Mostly for musicbrainz.org.

HOW TO INSTALL

Please install all your scripts from this GitHub page as versions you can find on GF and on OUJS might be out of date.

You can click the scripts to see their code then the Raw button to install them.

INSTALL-USER-SCRIPT adds a direct download/install icon button in front of each userscripts in github, gitlab and bitbucket file listings.

COMPATIBILITY

Here are setups that I use:

  • Firefox on Android (FF v68), Linux and Windows + Violentmonkey
  • Vivaldi on Windows + Violentmonkey

As Vivaldi is based on Chromium, it also works in Chrome. I have been told it also works in Safari. Please report bugs if it does not work in your setup (see next section).

ISSUES

If you encounter a problem, you can see if it is an already known issue and you can also upvote issues by clicking the thumb up icon on the issue description. Please eventually add an issue to report any bug.

※ Find more MusicBrainz scripts in Github, GF and OUJS.

※ Those scripts used to be hosted at jesus2099’s scripts - Userscripts.org (archived) since 18.7.2007 and until the May 2014 break down.

Here is a list of scripts that have not yet been stored here in “KONAMI COMMAND”:
