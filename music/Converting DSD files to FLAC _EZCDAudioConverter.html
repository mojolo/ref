Post by poiko Sun Feb 04, 2018 7:29 pm

FLAC Auto/Auto/Auto uses bit depth and sample rate from the DSD decoder output. It is DXD 32-bit floating point, 352.8kHz.

You should set the FLAC output to 24-bit 96kHz for DSD64 and 24-bit 192kHz for DSD128.

Re: Converting DSD files to FLAC

  • Quote

Post by poiko Mon Feb 05, 2018 12:33 pm

Auto/Auto/Auto uses source audio properties DSD decoder output to PCM that is DXD 32-bit float 352.8 kHz. FLAC files will be larger than the original DSD file.

It is because DSD uses very different (1-bit delta-sigma) coding than FLAC that uses PCM. There is no equal FLAC format for DSD. You should set FLAC to 24-bit 96 kHz for DSD64 and 24-bit 192 kHz for DSD128 to get equal audio quality. If you wish just to compress DSD to smaller files and retain the original DSD format then you can use the WavPack DSD output format.
