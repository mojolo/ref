    For moods, one person can write a script that extracts the song mood xml string from the allmusic's page for said song with the following regex (pretty dirty): (^.*)(?=allmusic)(.*)(?=mood)(.*)(>)(.*)(<)(.*)(<.*$)

    To grab the moods, have the above regex return $5 and somehow include them in the respective album tags.

    Yeah, I know I'm breaking a cardinal sin by using regex to parse html.   There must be other ways.

    EDIT:  Here's a better way with regex to selecting moods from a song's allmusic page: (?<=\b(allmusic.com/mood/)).*?(?=\-)
    Tip #2: If simply wants the capitalized moods in the song's AMG page, then here's the regex: Find: ^.*\b(allmusic.com/mood/).*">(.*)<\/a>.*$ Replace: \2 or $2

            Band_ID
            Album_ID
            Track#
            Title
            Track_ID
            Comments (or Description)
